        사전적 의미 : 배치
        기술적 의미 : 구성 요소들을 제한된 공간에 배치하는 기술

화면 배치 방법 : display

        요소가 화면에 어떻게 보여질지를 지정하는 속성
        - block : 화면을 수직 분할하는 방식(행으로 나눔)
        - inline : 화면을 수평 분할하는 방식(열로 나눔)
        - inline-block : 수직분할 + 수평분할
        - none : 화면에 요소를 보여주지 않도록 하는 방식
